# Afterlight Atlas

Afterlight Atlas is a cinematic Earth Day web experience for the DEV challenge prompt "Build for the Planet." Instead of acting like a guilt-heavy carbon calculator, it lets people compose speculative future worlds and feel the consequences through responsive visuals, narrative dispatches, system levers, and saved scenario snapshots.

## What makes it stand out

- Four original world seeds with radically different climate futures.
- Seven system levers covering power, mobility, food, water, materials, biodiversity, and civic care.
- A living planet scene that changes tone as the scenario changes.
- Derived "planet score" metrics that surface atmosphere relief, biodiversity return, civic resilience, water vitality, circular matter, and heat softening.
- A future dispatch engine that turns the chosen setup into story-rich scenario text.
- Judge Mode comparison against the seed baseline or a saved scenario.
- Present Mode for cleaner judge walkthroughs.
- Exportable judge comparison card as a PNG.
- One-click Best Future preset for instant high-performing scenarios.
- Three-horizon timeline cards for 2035, 2060, and 2085.
- Save, load, delete, copy, and JSON export flows for alternate futures.
- URL hash state so each world can be shared as a link.
- Mobile-friendly layout, custom branding, and social preview assets.

## Stack

- TypeScript
- Vite
- Handcrafted HTML rendering
- Custom CSS animation and layout system
- Local storage for saved snapshots

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Project framing

The experience is intentionally speculative. It is not a scientific climate model. Its goal is to make systemic planetary choices feel vivid, memorable, and discussable, which suits a creativity-focused Earth Day challenge entry.
